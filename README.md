# BloodBound: 2D Platformer
   For bored gamers who are looking for something fun in their spare time, without spending any money, Bloodbound is a new 2D platformer that is fully free and open source. Unlike other platformers from publishers like Nintendo, which come at a high cost, Bloodbound is completely accessible to all who wish to play or contribute. All needed info is down below.   

## How to run
1.) Install most recent java SE version
https://www.oracle.com/java/technologies/downloads/

2.) Download .jar file from most recent release page

3.) If you're are on mac run this command in your terminal: 
```bash
xattr -d com.apple.quarantine ~/Downloads/GameDemo-1.0.0.jar
```

4.) Double click on the jar and run!


_**Enjoy!**_

## How to contribute
Needed resources:
- Tiled (for map editing)
- LibGDX for the project
- Most recent JRE
- launch4j to bundle and release
